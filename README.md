
A modern, responsive e-commerce front-end built with React, Vite, Tailwind CSS, and more.
This project aims to provide a sleek shopping experience with product listings, cart functionality, and a clean UI.

 Features

Responsive layout for mobile, tablet and desktop

Product browse & search interface

Add to Cart / Remove from Cart experience

Tailwind CSS for fast styling

Vite for super-fast build & HMR

ESLint + Prettier configured for consistent code style


## Tech Stack






| Tool              | Purpose                     |
| ----------------- | --------------------------- |
| React             | UI library                  |
| Vite              | Build tool & dev server     |
| Tailwind CSS      | Utility-first CSS framework |
| ESLint & Prettier | Code linting & formatting   |
| JavaScript        | Primary scripting language  |





## Installation

1.Clone the repository

```bash
  git clone https://github.com/AJAY007-A/Ecommerce.git
  cd Ecommerce
```
2.Install dependencies

```bash
  npm install

```
3.Run the development server

```bash
 npm run dev

```
4.Build for production

```bash
 npm run build


```
